Automatic Graph Preview Tool for Online Datasets

Student: Fang Fei
Supervisor: Dr. Loretta Y.K. Choi
Department of Computer Science, The University of Hong Kong

Motivation

○    Open data is a foundational technology for smart city.

○    Open data portals aim to increase citizen participation in government.

Existing Problems:

 No or poor data preview functionality.

 Gaining insights from data files is time consuming.

 Existing visualization tools have a steep learning curve.

Aim

The main purpose of this project is to develop an Automatic Graph Preview Tool (AGPT), designed for open data portals like data.gov.hk, to provide a first glance at data from online datasets in a proper and easy-to-use manner.

Features

Recommendation Process

AGPT is empowered by a Hybrid Recommender, which suggests both what data to visualize (data query recommendations) and how to visualize it (visual encoding recommendations).

Construction of Diagrams

In AGPT, a diagram is defined by its Mark Type, Coordinate and Visual Variables. The Visual Variables include Positional Variables and Retinal Variables. The number of possible constructions of diagrams increases with the number of data fields needed to be encoded.
